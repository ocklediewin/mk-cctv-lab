# mk-cctv-lab

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。所有页面均为静态 HTML 文件，无后端依赖，可直接在浏览器中打开或通过 GitHub Pages 等静态托管服务部署。

## 项目简介

mk-cctv-lab 旨在收集、整理并保存一系列独立的 HTML 页面。这些页面各自独立，互不依赖，适合作为技术演示、功能测试、个人练习或特定场景下的参考素材。仓库本身不针对任何单一域名或具体网站，内容以通用性和可复用性为主。

## 目录结构

```
.
├── index.html          # 仓库首页 / 导航页面（可选）
├── page-01/            # 示例页面目录
│   └── index.html
├── page-02/
│   └── index.html
└── ...
```

- 每个独立页面通常放在一个以页面名命名的子目录中。
- 根目录下的 `index.html` 可作为总览或导航页（如存在）。
- 所有资源文件（CSS、JS、图片等）建议与 HTML 文件放在同一目录下，以保持自包含。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖外部服务器或数据库。
- 页面内容可能涉及多种主题，但均不包含对特定网站、服务或商业产品的指向性链接。
- 归档的页面可能不定期更新或新增，也可能因维护需要被移除或替换。
- 每个页面通常附带简单的注释或说明（位于 HTML 文件头部或 README 同级文档中），以帮助理解其用途。

## 使用方式

1. **本地查看**：直接克隆仓库，用浏览器打开任意 HTML 文件即可。
2. **在线部署**：可通过 GitHub Pages 或其他静态托管服务将仓库内容部署为网站。
3. **引用与修改**：可自由复制、修改或分发这些页面，但请保留原始注释（如有）。
